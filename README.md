# Язык ассемблера Учебной Виртуальной Машины (УВМ)

## Синтаксис команд

### LOAD_CONST <регистр> <константа>
Загружает 24-битную константу в регистр.
- `<регистр>`: номер регистра (0-31)
- `<константа>`: 24-битное целое число (0-16777215)

Пример: `LOAD_CONST 5 1000`

### READ_MEM <регистр_адрес> <регистр_результат>
Читает значение из памяти по адресу из регистра.
- `<регистр_адрес>`: регистр, содержащий адрес памяти
- `<регистр_результат>`: регистр для записи результата

Пример: `READ_MEM 1 2`

### WRITE_MEM <регистр_значение> <регистр_адрес>
Записывает значение из регистра в память по указанному адресу.
- `<регистр_значение>`: регистр с данными для записи
- `<регистр_адрес>`: регистр с адресом памяти

Пример: `WRITE_MEM 3 4`

### SHR <регистр> <адрес_памяти>
Выполняет побитовый логический сдвиг вправо.
- `<регистр>`: регистр с операндом и для результата
- `<адрес_памяти>`: адрес в памяти со значением сдвига

Пример: `SHR 5 256`

## Форматы команд

1. LOAD_CONST: 5 байт
   - Бит 0-6: код операции (44)
   - Бит 7-11: номер регистра (B)
   - Бит 12-35: константа (C)

2. READ_MEM: 3 байта
   - Бит 0-6: код операции (120)
   - Бит 7-11: регистр с адресом (B)
   - Бит 12-16: регистр результата (C)

3. WRITE_MEM: 3 байта
   - Бит 0-6: код операции (34)
   - Бит 7-11: регистр с данными (B)
   - Бит 12-16: регистр с адресом (C)

4. SHR: 6 байт
   - Бит 0-6: код операции (37)
   - Бит 7-11: номер регистра (B)
   - Бит 12-41: адрес в памяти (C)
